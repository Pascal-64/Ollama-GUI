# Prompt-Profile

## Profile

- `profiles/code_generate/` – neue Code-Erzeugung
- `profiles/code_debug/` – Fehlerbehebung mit Code als Ergebnis
- `profiles/code_refactor/` – Umstrukturierung mit Code als Ergebnis

Jedes Profil besitzt:
- `pipeline/` für 3 Stufen
- `oneshot/` für direkten Code-Output
